Python类中使用__call__

Jun 4, 2017

对象通过提供call(slef, [,*args [,**kwargs]])方法可以模拟函数的行为，如果一个对象x提供了该方法，就可以像函数一样使用它，也就是说x(arg1, arg2…) 等同于调用x.call(self, arg1, arg2) 。 模拟函数的对象可以用于创建防函数(functor) 或代理(proxy)
